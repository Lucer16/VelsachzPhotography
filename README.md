# Velsachz Photography 📸

Sitio web profesional de portafolio y servicios fotográficos desarrollado para **Josué Velasquez**, con sede en Penonomé, Coclé. 

Este proyecto destaca por su diseño minimalista, animaciones fluidas y una experiencia de usuario (UX) centrada en la conversión directa a través de canales de contacto rápidos.

## 🚀 Características Principales

* **Diseño Mobile-First:** Optimización total para dispositivos móviles con menús de acción rápida.
* **Estética Dark Mode:** Interfaz elegante utilizando una paleta de colores Negro, Blanco y Amarillo (#FACC15).
* **Componentes Interactivos:**
    * Hero Slider dinámico con transiciones de opacidad.
    * Sistema de revelado de secciones al hacer scroll (Reveal.js concept).
    * Marquesina animada para servicios.
* **SEO Avanzado:** Integración de Schema Markup (JSON-LD) para posicionamiento local en Panamá.
* **Tecnologías:** HTML5, Tailwind CSS, JavaScript (Vanilla).

## 🛠️ Stack Tecnológico

* [Tailwind CSS](https://tailwindcss.com/) - Framework de utilidades CSS.
* [Font Awesome](https://fontawesome.com/) - Iconografía profesional.
* [Google Fonts](https://fonts.google.com/) - Tipografías *Outfit* y *Plus Jakarta Sans*.
